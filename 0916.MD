# 기초웹개발론
------------



## Intellij IDEA



#### 프로젝트 생성

* spring-boot project
JDK Eclipse temurin을 다운받고 spring web 을 포함하여 프로젝트 생성
localhost:8080 에 출력


* html project
이름만 변경 후 프로젝트 생성
NVM(Node Version Manager) 설치
localhost:1234 에 출력
파일을 변경할 시 즉각적으로 웹또한 변경됨
